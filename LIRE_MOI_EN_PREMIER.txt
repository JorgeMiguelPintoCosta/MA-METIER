DETECTION DE DRONES - DEMARRAGE RAPIDE


Ce projet permet de détecter des drones à partir de signaux RF, en mode simulation ou en conditions réelles (avec SDR + GNU Radio).


CONFIGURATION DE BASE


1. Installer Python (3.9 ou plus récent) et les dépendances :
   pip install -r requirements.txt

2. Configurer la base de données PostgreSQL si besoin :
   - Remplir le fichier .env avec les bons identifiants (DB_NAME, DB_USER, DB_PASSWORD, DB_HOST, DB_PORT)


MODE SIMULATION (par défaut)


- Par défaut, le projet fonctionne en mode simulation (pas besoin de SDR ni de GNU Radio).
- Les signaux sont générés artificiellement pour tester toute la chaîne de détection.
- Pour lancer :

    python main.py


PASSER EN MODE REEL (avec SDR + GNU Radio)


1. Ouvrir le fichier rf_input/source.py
2. Remplacer la ligne :

    USE_GNURADIO = False

   par

    USE_GNURADIO = True

3. Lancer le script Python normalement :

    python main.py

Le programme recevra alors les signaux envoyés en temps réel (via TCP, port 5000).

CONSEILS ET DEPANNAGE


- Si rien ne s'affiche, vérifier la configuration du port et le type de données (voir README).
- Pour ajuster la sensibilité ou les seuils de détection, modifier les paramètres dans les modules de détection.
- Pour tester la connexion à la base de données uniquement :

    python test_database.py
